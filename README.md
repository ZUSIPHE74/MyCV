# MyCV

This project is a static website for Practical Task 1, demonstrating deployment using GitHub Pages.

## Project Structure
- `index.html`: Main homepage (CV)
- `styles.css`: Stylesheet for the site
- `static_site_example_index.html`: Example static site for deployment demonstration

## How to View
Visit the deployed site at:
https://zusiphe74.github.io/MyCV

## GitHub Pages Deployment Steps
1. Created a public repository and initialized with a README.
2. Uploaded all project files, keeping the structure intact.
3. Activated GitHub Pages from repository settings (main branch).
4. Waited for deployment and accessed the site via the provided URL.
